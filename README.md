Deteksi Kendaraan Menggunakan YOLOv8

Deskripsi
Proyek ini merupakan implementasi Computer Vision menggunakan YOLOv8 untuk mendeteksi berbagai jenis kendaraan pada gambar maupun video. Model dilatih menggunakan dataset Road Vehicle Images Dataset dari Kaggle dan dijalankan menggunakan Google Colab.

Fungsi/Fitur
Mendeteksi kendaraan secara otomatis pada gambar atau video.
Mengenali beberapa jenis kendaraan, seperti:
1.Car
2.Bus
3.Truck
4.Motorcycle

Menampilkan bounding box pada setiap kendaraan.
Menampilkan nama kelas kendaraan.
Menampilkan confidence score hasil deteksi.
Mendukung pengujian menggunakan gambar maupun video.
